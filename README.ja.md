# genstoryai

## プロジェクト概要

genstoryaiは、完全なフロントエンドとバックエンドを備えたAI支援小説生成アプリケーションです。
ユーザーはキャラクター、背景、ストーリーライン（並行、統合、分割可能）、重要な瞬間、執筆スタイルの選択、各章の充実などを行うことができます。

## 機能特性
- FastAPIによる高性能API
- Pydanticによるデータ検証とモデル定義
- Shadcnデザインシステム
- 設定可能な大規模言語モデル
- SQLiteストレージ
